# Flower Store

## API-интерфейс FlowerStore
Работает на порту `10001`.

`DbContext` можно настроить на один из серверов `MSSQL`, 
`PostgreSQL`, `MySQL`, `SQLite` через `appsettings.json`.

При первом запуске в базе создается логин администратора:

Логин: `admin`, Парол: `@dmin123$`
